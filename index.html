gallifreyan.js

A circular gallifreyan translator

circular gallifreyan is a beautiful language designed by Loren Sherman at shermansplanet. This is an html5 based transliterator that can turn text into circular gallifreyan and render to a canvas or a C2S SVG context.

This is nowhere near complete so expect many things to be broken.

This project is maintained by markrileybot
